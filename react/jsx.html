# JSX语法糖

React 使用 JSX 类替代常规的 JavaScript。JSX 看起来像 XML 的JavaScript 语法的扩展，在 React 中会被babel编译为JavaScript。

# JSX的特点

  1. JSX执行更快，因为它在编译为JavaScript代码后进行了优化。
  2. 它是类型安全的，在编译过程中就能发现错误。
  3. 使用 JSX 编写模块更加简单快速。

# JSX知识小记

# JSX注意几个书写格式

  1. React 的JSX是使用大写和小写来区分是组件还是 HTML标签。
  2. JSX和HTML的标签属性的区别
HTML标签属性 JSX 原因
for htmlFor for在JS中为for循环关键字
class className class在JS中为声明类关键字
style 需使用JS对象(使用双花括号{{}})

示例：

import React, { Component } from 'react'
class App extends Component{
    render(){
        return(
            <div>
                <p style={{color: "red", fontSize: 30}}>JSX 标签属性</p>
                <p className="box">className属性</p>
                <label htmlFor="username">
                  用户名：<input id="username" type="text" />
                </label>
            </div>
        )
    }
}

# 在JSX中使用变量，三目运算和循环

在 JSX 中使用变量，需要在 return中直接使用单花括号{}调用，示例如下:

import React, {Component} from "react"

let name = "落枫", age = 18, list = [10, 20, 30, 50]
export default class App3 extends Component{
    render(){
        return(
            <div>
                <p>姓名: {name}</p>
                <p>年龄: {age}</p>
                <p>是否已经成年：{age >= 18 ? '是' : '否'}</p>
                <ul>
                    {
                        list.map((value, key) => {
                            return(
                                <li key={key}>{value}</li>
                            )
                        })
                    }
                </ul>
            </div>
        )
    }
}

# 实现计数器累加案例

实现点击按钮，计数器累加案例,需要定义一个变量保存累加的结果，因此需要定义一个组件的状态数据

import React, { Component } from 'react'
export default class App4 extends Component {
    // 构造函数 接收参数
    // 创建组件时会执行构造函数
   
    constructor(props){
        super(props)  // 如果类继承另外一个类 需要在构造函数中加上 super()
        this.state = { // 定义一个组件的状态数据
            num: 20
        }
    }

    handleClick(){
        console.log('点击了按钮')
        // 写法和微信小程序类似
        // this.state.num++ 此写法不会生效
        this.setState({
            num: this.state.num+1
        })
    }

    render() {
        return (
        <div>
            {/* 
                <button onClick={this.handleClick}>按钮</button> 
                如果这么写或报如下错误： 
                Cannot read properties of undefined (reading 'setState')  
            */}
            {/* 需要将事件绑定到 this 上 */}
            <button onClick={this.handleClick.bind(this)}>按钮</button>
            {/* 如果使用箭头函数 就不需要bind(this) */}
            {/* <button onClick={ ()=> this.handleClick()}>按钮</button> */}
        </div>
        )
    }
}

# 实现数据双向绑定

import React, { Component } from 'react'
// 双向数据绑定
export default class App5 extends Component {

    constructor(props){
        super(props)
        this.state = {
            val: '双向绑定'
        }
    }

    handleChange(e){
        console.log(e.target.value);
        // 需要手动修改变量的值
        this.setState({
            val: e.target.value
        })
    }

    render() {
        return (
        <div>
            <input type="text" value={ this.state.val } onChange={this.handleChange.bind(this)} />
            <p>{ this.state.val }</p>
        </div>
        )
    }
}

# tab栏切换案例

import React, { Component } from 'react'
// tab 栏案例
import './assets/css/tab.css'
export default class App6 extends Component {

    constructor(props){
        super(props)

        this.state = {
            num: 1
        }
    }

    handleClick(number){
        console.log('点击了按钮', number)
        this.setState({
            num: number
        })
    }

    render() {
        return (
        <div className='tab_con'>
            <div className='tab_btns'>
                <input type="button" className={ this.state.num === 1 ? 'active' : ''} value="按钮1" onClick={this.handleClick.bind(this, 1)} />
                <input type="button" className={ this.state.num === 2 ? 'active' : ''} value="按钮2" onClick={this.handleClick.bind(this, 2)} />
                <input type="button" className={ this.state.num === 3 ? 'active' : ''} value="按钮3" onClick={this.handleClick.bind(this, 3)} />
            </div>
            <div className='tab_cons'>
                <div className={ this.state.num === 1 ? 'current' : ''}>按钮1对应的内容</div>
                <div className={ this.state.num === 2 ? 'current' : ''}>按钮2对应的内容</div>
                <div className={ this.state.num === 3 ? 'current' : ''}>按钮3对应的内容</div>
            </div>
        </div>
        )
    }
}

# 事件使用注意

1. 构造函数 (constructor) 接收参数 props
2. 如果类继承另外一个类 需要在构造函数中加上 super(props)
3. 更改变量值:

  • 需要定义组件的状态数据

  • 改变变量值的写法与小程序类似

    this.setState({
    num: this.state.num+1
})
    1
    2
    3

4. 绑定事件时需要将事件绑定到 this 上，如：

 this.handleClick.bind(this)
1

否则会报如下错：

错误提示

    Cannot read properties of undefined (reading 'setState')

5. 绑定事件时可以使用箭头函数

{/* 如果使用箭头函数 就不需要bind(this) */}
<button onClick={ ()=> this.handleClick()}>按钮</button>
