# React特点

1. 声明式设计

react 是面向数据编程，不需要直接去控制DOM，只需要把数据操作好，react会去操作DOM，可以节省很多操作DOM的代码，这就是声明式开发

2. 使用JSX语法

JSX是JavaScript语法的扩展。react开发大部分使用JSX语法

3. 灵活

react所控制的DOM就是id为root的dom,页面上的其他dom元素也可以使用JQ等其他框架

5、组件化开发

​ 通过 React 构建组件，使得代码更加容易得到复用和维护，能够很好的应用在大项目的开发中。

6、单向数据流

​ react是单向数据流，父组件传递给子组件的数据，子组件能够使用，但是不能直接通过this.props修改。 这样让数据清晰代码容易维护。

# 编写自己的第一个React

在上述基础中

  1. 删除 src 目录下的所有文件
  2. 在 src 目录下新建 index.js 文件，react项目的入口文件，编写如下代码：
// 1、导入核心代码
import React from "react"
import ReactDOM from "react-dom"
// 2、把对应的内容渲染到id 为root的标签上 (/public/index.html)

/**
 * ReactDOM.render(param1, param2)
 * param1: 作为 root 的内部内容；<div>Hello 落枫</div>
 *      注意： param1 根标签只有一个
 *                   这种语法不是字符串也不是* * HTML，这就是JSX, 即js的扩展语法
 * 
 * param2: 指向需要渲染到 /public/index.html 中的哪个标签(root)
 *             
 */
ReactDOM.render(
    <div>Hello 落枫</div>,
    document.getElementById('root')
)

// 在页面上会渲染出： Hello 落枫

  1. 组件化开发

index.js 是React项目的入口，需要保持文件的简洁，所以将 param1 的内容封装到 App.js 中。

在 src 目录下新建一个 App.js 文件，编写如下代码：

// 组件化开发
// 1、引入核心代码
import React from "react"
// import ReactDOM from "react-dom" 不需要引入，渲染内容只需要引入一次即可(index.js)

// 2、定义一个组件
class App extends React.Component{
    render(){
        return (
            <div>Hello 这是第一个组件</div>
        )
    }
}

// 3、导出组件
export default App

import ReactDOM from "react-dom"

    <App />,
    document.getElementById('root')

// 在页面上会渲染出： Hello 这是第一个组件

# 快捷生成一个 React 组件

要定义React组件,需要扩展 React.Component， 在 Vscode 扩展中搜索：react，找到 ES7+ React/Redux/React... 安装即可；然后创建名为：App.js 的文件,输入: rcc即可生成如下代码：

import React, { Component } from 'react'
export default class App4 extends Component {
  render() {
    return (
      <div>App4</div>
    )
  }
}

# 创建React项目小结

小结

  1. index.js 是React项目的入口文件, 文件名不能改变
  2. <div>Hello 这是第一个组件</div> 这种写法不是字符串也不是HTML, 这就是JSX, JS的一种扩展语法
  3. 保持入口文件的简洁，组件化开发
  4. 组件首字母大写
  5. 定义组件的时候，return 后面只能有一个根标签，但是这个标签内部可以有多个标签

快捷小技巧

在setting.json中配置： "emmet.triggerExpansionOnTab": true // 开启按Tab 键补全代码

注意：

如果在index.js 中param1参数直接写为：<div>Hello 这是第一个组件</div> 编译错误，需要在setting.json中配置：
"emmet.includeLanguages": { "javascript": "javascriptreact" },
