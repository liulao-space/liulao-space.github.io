# 不可控组件和可控组件

大多数情况下，使用受控组件来实现表单。 不清楚使用那种控件，阅读这篇文章了解受控组件和不受控组件 (opens new window)

# 不受控组件

组件中表单元素没有写value值，与state状态数据无关，不受组件管理。(不推荐)

使用一个 ref 来从DOM获得表单的值。

如果需要设置默认值，使用defaultValue属性 而不是 value属性。同样 type="radio" 或 type="checkbox" 设置默认值使用 defaultChecked 属性

import React, { Component } from 'react'
// 表单元素 不受控组件 表单数据由 DOM 元素本身处理
export default class App extends Component {

  handleClick(){
      // 使用 ref 来从DOM 获得表单值
      console.log(this.refs.username.value);
      console.log(this.refs.password.value);
  }

  render() {
    return (
      <div>
        {/* 这种组件和组件本身state数据没有关系，所以不受组件管理，称为不受控组件 */}
        {/* 这种写法的每个表单元素的验证在点击登录按钮时完成，用户体验差 */}
        用户名：<input type="text" defaultValue="落枫" ref="username" /> <br /><br />
        密&emsp;码：<input type="password" ref="password" /><br /><br />
        <input type="button" value="登陆" onClick={this.handleClick.bind(this)} />
      </div>
    )
  }
}

# 受控组件

组件中的表单元素的value值受组建的state数据控制，并且该表单元素有onChange 事件，可以在事件中对表单进行实时验证. 推荐

import React, { Component } from 'react'
// 表单元素 受控组件
export default class App extends Component {

  constructor(props){
    super(props)
    this.state = {
      username: '',
      password: ''
    }
  }

  handleChange(e){
      this.setState({
        username: e.target.value
      })
      // 可以编写需求逻辑
  }

  render() {
    return (
      <div>
        用户名：<input type="text" value={this.state.username} onChange={this.handleChange.bind(this)} /> <br /><br />
        密&emsp;码：<input type="password" /><br /><br />
        <input type="button" value="登陆" />
      </div>
    )
  }
}

# 处理多个输入元素

如果需要处理多个受控的 input 元素时，可以给每个元素添加一个 name 属性，并且在 onChange 事件处理函数中根据 event.target.name 的值来选择做什么。

import React, { Component } from 'react'
// 表单元素 受控组件处理多个 input 元素
export default class App3 extends Component {

    constructor(props){
        super(props)

        this.handleChange = this.handleChange.bind(this)
        this.handleSubmit = this.handleSubmit.bind(this)

        this.state = {
            username: '',
            password: '',
            sex: 1
        }
    }

    handleChange(e){
        let{ name, value} = e.target
        console.log(name + ': ' + value)
        // 根据表单设置的 name 属性来改变表单元素的值
        this.setState({
            [name]: value
        })
    }

    handleSubmit(){
        console.log('提交表单数据：', this.state)
    }

    render() {
        return (
        <div>
            用户名：<input type="text" name="username" onChange={this.handleChange} /> <br /><br />
            密&emsp;码：<input type="password" name="password" onChange={this.handleChange} /> <br /><br />
            性&emsp;别：男<input type="radio" name="sex" defaultChecked value={1} onChange={this.handleChange}  /> 女<input type="radio" name="sex" value={2} onChange={this.handleChange} /><br /><br />
            <input type="button" value="登陆" onClick={this.handleSubmit} />
        </div>
        )
    }
}

# 小结

React 中的表单组件分为 2 类：

# 1. 不受控组件（和状态无关）

1.1 在input 标签组件中，使用 ref 属性，来代表元素标识，获取元素的值 1.2 在组件渲染后，会自动把有ref属性的DOM对象，挂载到 this.refs上

this.refs = {
   ref名1: DOM 对象,
   ref名2: DOM 对象,
}
