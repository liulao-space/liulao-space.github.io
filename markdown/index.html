# 基本语法

# 标题语法

在单词或则短语前面添加井号(#), #号的数量代表标题的级别(h1 - h6), 及 1 - 6 个#。

  • 效果如下：

    • # #一级标题

    • # ##二级标题

    • # ###三级标题

    • # ####四级标题

    • # #####五级标题
    • # ######六级标题

# 段落

要创建段落，使用一行或多个空白行分割。

# 换行

在一行的末尾添加两个或多个空格，然后回车即可。也可以使用<br>标签

# 强调语法

# 粗体

需要加粗文本，在文本前后各添加两个星号(**) 或 两个下划线 (__)即可。

这是使用两个 __ 加粗的文本

# 斜体

要用斜体文本时，在文本前后各添加一个星号(*) 或 下划线(_)即可。

注意

文本前后不能带空格，需紧跟星号或者下划线

# 列表语法

# 有序列表

在每个列表项前面添加数字并且紧跟一个英文句点( . )。数字不必按数学顺序排序，但是列表应当以数字 1 开始。

  1. 落枫1
  2. 落枫2
  3. 落枫3
  4. 落枫4
  5. 落枫5

# 无序列表

在每个列表列表项前面添加破折号( - )、星号( - )或加号( + ), 缩进一个或者多个列表项可创建嵌套列表。
