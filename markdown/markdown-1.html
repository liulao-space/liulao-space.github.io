# 表格

添加表，使用三个或多个连字符 --- 创建每列的标题，并使用 | 分隔每列

# 对齐

您可以通过在标题行中的连字符的左侧，右侧或两侧添加冒号（:），将列中的文本对齐到左侧，右侧或中心。

| 姓名      | 性别         | 年龄     |
| :---      |    :----:    |    ---: |
| 落枫      | 男           | 18       |
| 王二麻子  | 男           | 28       |

# 任务列表语法 【使用无效过】

可以创建带有复选框的项目列表。要创建任务列表，需在任务项之前添加破折号 - 和 方括号 [],并在 [] 前面加上空格。要选择一个复选框，需要在 [] 之间加一个 x, 即[x]。

  • [x] Write the press release
  • [ ] Update the website
  • [ ] Contact the media
